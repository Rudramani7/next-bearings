# Next.js App Router + Tailwind + DummyJSON Assessment

## Quick start
1. Install dependencies:
   npm install
   npm install -D tailwindcss postcss autoprefixer

2. Run dev:
   npm run dev

## Notes
- Uses DummyJSON API for product data: https://dummyjson.com
- App Router with SSR for product pages (force-dynamic)
- Cart is client-side persisted (localStorage)
- ESLint configured (next/core-web-vitals)
